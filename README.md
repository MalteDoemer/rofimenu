# ROFIMENU

This is a small python project that uses [rofi](https://github.com/davatorium/rofi) to display menus.


## Installation

Clone the repository:

```git clone https://github.com/MalteDoemer/rofimenu.git```

Change directory:

```cd rofimenu```

Install it with pip:

```pip install .```
